# 🚀 README — Projecte Inicial SMX 2n

## 📘 Descripció del projecte

Benvinguts i benvingudes al primer projecte del segon curs de **Sistemes Microinformàtics i Xarxes** de l’Escola Pia Mataró.

Aquest projecte marca l’inici de la vostra etapa com a futurs professionals. A partir d’ara, treballareu amb una metodologia real, orientada al món laboral.

---

## 🎯 Objectius del projecte

Aquest projecte no té com a objectiu crear un producte final complet.

En lloc d’això, es busca:

- Aprendre a treballar amb una **metodologia professional**
- Organitzar tasques de manera eficient  
- Desenvolupar habilitats de treball en equip  
- Entendre com funciona un projecte real  

👉 El veritable producte és el **vostre aprenentatge**.

---

## ⚙️ El repte

En aquest primer projecte es vol que:

- Treballeu de manera organitzada i constant  
- Utilitzeu metodologies àgils com **Kanban**  
- Practiqueu amb eines digitals com **Microsoft Planner**  
- Desenvolupeu un sistema de treball estructurat  

---

## 🧩 Rol del Projecte Intermodular

El mòdul de **Projecte Intermodular** actuarà com a client:
