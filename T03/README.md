# 🌐 README — Fonaments de Hosting i Naming

## 🎯 Objectiu general

L’objectiu d’aquesta activitat és **entendre els fonaments tècnics** necessaris per publicar i mantenir un lloc web en entorns reals, així com desenvolupar habilitats per escollir serveis de hosting i crear noms de marca adequats.

***

# 🧩 T03 — Part 1: Escollir el millor hosting

## 🎯 Objectiu

Aprendre a:

* Analitzar les necessitats d’un projecte web
* Triar un servei de hosting adequat
* Justificar la decisió amb criteris tècnics, comercials i de servei

***

## 📚 Casos proposats

Cada alumne ha d’escollir (o se li assigna) **un cas** i fer-ne l’anàlisi.

### 🛒 Cas 1 — Botiga online

* Tràfic: 5.000 – 20.000 visites/mes
* Ubicació: Espanya i Europa
* Requisits:
  * Processament de pagaments
  * Moltes imatges i vídeos
  * SSL obligatori
  * Suport en castellà/català
  * Creixement previst del 50–100%

***

### 📰 Cas 2 — Blog / revista digital

* Tràfic: 1.000 – 5.000 visites/mes
* Ubicació: Amèrica Llatina (80%)
* Requisits:
  * Streaming (podcasts i vídeo)
  * Backups freqüents
  * Domini .com
  * Cost moderat
  * Escalabilitat

***

### 🔐 Cas 3 — Aplicació web ONG

* Tràfic: 500 – 2.000 usuaris mensuals
* Ubicació: Catalunya / Espanya
* Requisits:
  * Alta seguretat
  * Compliment RGPD
  * Base de dades robusta
  * Backups automàtics
  * Bon suport tècnic

***

## 🏢 Empreses de hosting a comparar

* Webempresa
* Raiola Networks
* SiteGround
* Aruba.it

*(Opcional: Hostinger, IONOS, LucusHost...)*

***

## 📊 Metodologia

### ✅ Passos a seguir

1. Analitzar el cas

2. Definir necessitats:
   * Tràfic
   * Seguretat
   * Ubicació
   * Creixement
   * Pressupost

3. Definir criteris:
   * Espai de disc
   * CPU / RAM
   * SSL
   * Backups
   * Preu
   * Escalabilitat
   * Suport tècnic

4. Investigar 4 proveïdors

5. Crear una taula comparativa

***

# 🧩 T03 — Part 2: Naming i dominis

## 🎯 Objectiu

Aprendre a:

* Crear noms de marca
* Validar dominis i xarxes socials
* Analitzar qualitat del naming

***

## 📝 Tasca

Cada alumne haurà de:

1. Escollir 1 cas (dels 20 proposats)

2. Crear **3 noms de marca**

3. Validar-los a **namecheck.com**:
   * Domini
   * Xarxes socials

4. Fer captures de pantalla

5. Analitzar cada nom amb ChatGPT

6. Crear document final amb conclusions

***

## 🤖 Prompt per validar el naming

```
Vull que actuïs com un crític dissenyador web expert en naming. 
Avalua el nom amb aquests criteris:

1. Claredat  
2. Originalitat  
3. Brevetat  
4. Pronunciació  
5. Idioma  
6. Domini  
7. Xarxes socials  
8. Escalabilitat  
9. Connotacions negatives  
10. Impacte de màrqueting  

- Dona comentaris breus
- Dona puntuació 0–100
- Proposa millores

Nom: [nom del projecte]  
Descripció: [explicació del projecte]
```

***

## 🎲 Propostes de casos (exemples)

* Botiga de torradores vintage
* ONG per a hàmsters obesos
* Marketplace de mitjons desaparellats
* App per gestionar formigues
* Plataforma per parlar amb plantes
* Blog de receptes blaves
* Club de ventiladors retro

*(i més opcions del llistat)*

***

## 📦 Entregable final

Document amb:

* Cas escollit
* 3 noms creats
* Captures de:
  * namecheck.com
  * anàlisi de ChatGPT
* Justificació final del nom escollit
