# 🔐 README — Introducció a la Seguretat Informàtica

## 📘 Context

Tot just comencem el segon curs i una de les principals preocupacions és la realització de l’**estada a l’empresa**.

En aquest escenari, imagineu que heu rebut una **oferta de pràctiques** al departament de **seguretat informàtica** d’una empresa.

Tot i la il·lusió, sou conscients que la seguretat informàtica és un àmbit encara poc conegut per vosaltres, per això decidiu començar a investigar els conceptes bàsics per preparar-vos per a una possible entrevista.

---

## 🎯 Objectiu de la tasca

Aquesta primera activitat té com a objectiu:

- Introduir els **conceptes bàsics de la seguretat informàtica**
- Servir de base per a temes més avançats que es treballaran posteriorment
- Preparar-vos per a situacions reals, com una entrevista de feina
